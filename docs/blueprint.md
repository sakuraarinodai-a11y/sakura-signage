# **App Name**: Kumite Score

## Core Features:

- Match Score Display: Real-time, prominent display of current scores for two contestants.
- Point Awarding: Clear buttons to award common karate points (e.g., Ippon, Waza-ari) to either contestant with immediate score update.
- Match Timer: A precise countdown timer for the duration of a round.
- Round Management: Controls to start, pause, reset the timer, and advance to the next round, along with a round counter.
- Contestant Naming: Input fields to enter and display the names of the two competing martial artists.
- Match Initialization: A single function to clear all scores, reset the timer, and reset the round number to prepare for a new match.
- AI Rule Explainer: An AI-powered tool that answers natural language questions about specific kumite rules or clarifies point awarding scenarios based on official WKF guidelines (acting as a knowledge base).

## Style Guidelines:

- Primary color: Deep indigo (#261A66), reflecting discipline and focus, to highlight interactive elements and main score display.
- Background color: Very dark charcoal with a hint of purple (#17151A), providing a sophisticated and low-distraction backdrop for intense matches.
- Accent color: Vibrant sky blue (#268ED9), offering high contrast and used for key action buttons and status indicators.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring legibility of scores and names.
- Use minimalist, high-contrast, filled icons for all action buttons (e.g., score increase, start/pause), ensuring immediate recognition and clear calls to action.
- A clean, dual-panel layout, separating each contestant's score and controls, with the timer prominently displayed centrally. Large, bold numbers for scores are essential.
- Subtle visual feedback for score updates (e.g., a brief flash or minor scaling of the score). Smooth transitions for timer countdown and round changes to maintain flow.
